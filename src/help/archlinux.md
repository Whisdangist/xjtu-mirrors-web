# Archlinux 镜像使用帮助

## 收录架构

- i686

- x86_64

## 收录版本

- ALL

## 使用说明

编辑/etc/pacman.d/mirrorlist，在文件的最顶端添加


```
Server = http://mirrors.xjtu.edu.cn/archlinux/$repo/os/$arch
```
